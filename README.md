deviceServer
====


1, devicesServer.py 负责接收与处理设备上传信号
2, missionPlannerServer.py 负责接收 MP 地面站的请求
   a, 并返回当前所有设备 IP 列表
   b, 注册希望控的制设备
   c, 透传设备上传的信息
   d, 下发设备控制信息
